# NFT-Music-Player
We build a Web3 NFT music player with functionalities like music trading, searching for songs and joining membership.

[Link](https://www.youtube.com/watch?v=FsixRtyymDc) to demo

## Technology & Tools
- Javascript (React & Testing)
- Solidity (Writing Smart Contract)
- [Ethers](https://docs.ethers.io/v5/) (Blockchain Interaction)
- [Hardhat](https://hardhat.org/) (Development Framework)
- [Ipfs](https://ipfs.io/) (Metadata storage)
- [React](https://reactjs.org/) (User Interface development)
- [React routers](https://v5.reactrouter.com/) (Navigational components)

## Functionalities
- Buy tokens with Metamask wallet
- Resell Tokens with any price
- Play music before purchasing it
- Search Box where you can search tokens with a name
- See your listed and sold tokens
- Join membership
